# Referências - Contract Analyzer

Este documento lista todas as referências e recursos utilizados durante o desenvolvimento do projeto **Contract Analyzer**. As referências estão organizadas por categorias para facilitar a consulta.

## 1. Bibliotecas e Frameworks Utilizados

### 1.1. Backend
- **ASP.NET Core**: Framework principal utilizado para o desenvolvimento do backend.
  - [Documentação Oficial do ASP.NET Core](https://docs.microsoft.com/en-us/aspnet/core/?view=aspnetcore-6.0)
- **MongoDB C# Driver**: Driver oficial do MongoDB para comunicação com o banco de dados.
  - [MongoDB C# Driver Documentation](https://mongodb.github.io/mongo-csharp-driver/)
- **AWS SDK for .NET**: SDK para integração com serviços AWS, como S3.
  - [AWS SDK for .NET Documentation](https://docs.aws.amazon.com/sdk-for-net/v3/developer-guide/welcome.html)
- **SpacySharp**: Biblioteca de processamento de linguagem natural para .NET, baseada no Spacy.
  - [SpacySharp GitHub Repository](https://github.com/some-repo/spacysharp)

### 1.2. Frontend
- **Angular**: Framework de front-end utilizado para construir a interface do usuário.
  - [Angular Documentation](https://angular.io/docs)
- **Angular Material**: Biblioteca de componentes baseada no Material Design para Angular.
  - [Angular Material Documentation](https://material.angular.io/)
- **Tailwind CSS**: Framework de CSS utilitário para estilização personalizada.
  - [Tailwind CSS Documentation](https://tailwindcss.com/docs)
